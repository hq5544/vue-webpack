# 行
##传入参数

```js
props: [
  'link',
  'hide'
]
```

- link 点击跳转地址
	- 对象
	- vue-router 格式对象
- hide 是否展示
	- 布尔

## 样式
两端对齐。
白色背景。
上下 1px 线框。
![-w300](media/14617459933347/14617494980840.jpg)


## 用法
组件引入。
三个 slot：默认、right、unit（单位）。
未插入 slot 舍弃。
可嵌入 div、input、select、codeBtn。
插入单独 input 元素，自动出现清空按钮。
