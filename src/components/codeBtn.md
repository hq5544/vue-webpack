# 验证码按钮
##传入参数

```js
props: [
  'phoneNum',
  'required',
  'fn'
]
```

- phoneNum 电话号码
	- 字符串
- required 是否必须电话号码
	- 布尔
	- 如果必须，必须传入电话号码，号码不通过会弹窗提醒
	- 如果非必需，则不传入不验证
- fn 点击按钮触发函数
	- 函数
