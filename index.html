介绍

介绍

这是一个基于 vue 官方 webpack 模版进行修改的项目。从我们公司实际项目中剥离出来的基本组件、工具、和基本配置的集合。

项目地址：vuejs-templates/webpack

这个项目只包含了基础的组件，很少涉及样式。 今后还会继续添加组件库。

使用了大量 ES6 语法, 还有 airbnb 的 eslint 标准。

快速开始

使用这个模版，你需要全局安装好 vue-cli。 推荐使用 npm 3+ 来构建更加高效的依赖树。

$ npm install -g vue-cli
$ vue init hq5544/vue-webpack my-project
$ cd my-project
$ npm install
$ npm run dev
