介绍

基础组件

该模版自带了一部分基本组件，效果可以在 demo 页查看。

组件列表

  • row 行基础组件，不直接使用
  • ele 行组件，基于 row 组件
  • group 组组件
  • btn 按钮组件
  • codeBtn 验证码按钮组件，基于 btn 组件
  • freeBtn 自由拖动按钮组件，基于 btn 组件
  • backdrop 遮罩层组件
  • alert 弹窗基础组件，不直接使用
  • popup 弹窗组件，基于 backdrop 和 alert 组件
  • loading 进度条组件
  • more 加载更多组件
  • cityList 地址选择组件
